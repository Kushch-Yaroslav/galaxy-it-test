# Contracts Platform Test Task

Test project built with Next.js, TypeScript, and CSS Modules.

## Features

- Responsive landing page based on the provided design
- Reusable component architecture
- Contracts carousel with drag-to-scroll
- SSR contracts loading
- API routes with mock JSON data
- Search page with query params
- Search API with substring matching
- Animated sections using Framer Motion

## Tech Stack

- Next.js
- TypeScript
- CSS Modules
- Framer Motion

## Installation

```bash
pnpm install
```

## Environment

Create a `.env.local` file in the project root:

```env
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

## Run

```bash
pnpm dev
```

Open `http://localhost:3000`.
